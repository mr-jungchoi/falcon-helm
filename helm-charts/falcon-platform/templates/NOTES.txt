## Component Status

Check deployment status with:
  kubectl get pods -n {{ .Values.global.namespace | default "falcon-system" }}

Monitor components:
  kubectl logs -n {{ .Values.global.namespace | default "falcon-system" }} -l app.kubernetes.io/managed-by=falcon-platform
